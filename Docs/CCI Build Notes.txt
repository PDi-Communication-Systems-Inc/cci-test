Building and installing CCI Test

 1) build engr build of full Android OS
 
 2) in the system/cci folder on the linux box:
     $ mm
 
 3) copy from linux box
     ~/pdi_android_bcm_solo/out/target/product/sabresd_6dq/system/bin/cci
    to anywhere on pc
 
 4) from windows prompt:
     > adb remount
 
  NOTE: for TVRC, step 4 does not work.  You must follow these steps from windows prompt
        then continue with step 5:
   > adb shell
   $ su
   # mount -o remount,rw /system
   # chmod 775 /system/bin
   # exit
   $ exit


 5) copy cci from pc "PATH" to target with "adb push" on windows command line as shown
     > adb push PATH\cci /system/bin
 
 6) from console shell on Android target:
     $ cd /system/bin
     $ chmod 766 cci
